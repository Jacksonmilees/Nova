# hello_world

A simple Python application that prints "Hello, World!" and displays the current time.

## Usage

1. **Clone the repository:**  `git clone <repository_url>` (If applicable)
2. **Navigate to the directory:** `cd hello_world`
3. **Run the application:** `python main.py`

This will print "Hello, World!" followed by the current date and time to your console.


## Dependencies

This application has no external dependencies.  It uses only Python's built-in `datetime` module.
